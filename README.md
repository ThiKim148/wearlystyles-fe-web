# PnBook - Frontend (wearlystyless-fe-web)

The frontend project for the PnBook system, is built with React 19, Vite and TypeScript.

## Tech Stack

- **Core:** React 19, TypeScript, Vite
- **Styling:** TailwindCSS v4, clsx, tailwind-merge
- **State Management:** React Query (TanStack Query)
- **Forms:** React Hook Form
- **Code Quality:** ESLint (Flat Config), Prettier, Husky, Lint-staged, Commitlint

## Setup requirement

- Node.js >= 18
- Package Manager: npm/pnpm/yarn
